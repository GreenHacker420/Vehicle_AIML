# Backend — FleetAI

FastAPI backend for vehicle maintenance risk prediction.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate

# Core runtime deps
pip install -r requirements.txt

# Optional: GenAI/LangChain layer (only needed when ENABLE_LLM_INSIGHTS=true)
pip install -r requirements-llm.txt

# Dev / test deps
pip install -r requirements-dev.txt

PYTHONPATH=backend uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## Endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/health` | Liveness probe |
| `GET` | `/metrics` | Model evaluation metrics (generated by `model/train.py`) |
| `POST` | `/predict` | Predict maintenance risk — JSON or CSV upload |
| `POST` | `/vehicles` | Register a new vehicle |
| `GET` | `/vehicles` | List all registered vehicles |
| `GET` | `/vehicles/{id}` | Get a specific vehicle by id |

## Configuration

Environment variables (prefixed with `FLEET_`):

| Variable | Default | Description |
|---|---|---|
| `FLEET_MODEL_PATH` | auto-discovered | Path to trained model `.pkl` |
| `FLEET_CORS_ORIGINS` | `["*"]` | Allowed CORS origins |
| `FLEET_DEBUG` | `false` | Enable debug logging |

Optional GenAI variables (only read when `ENABLE_LLM_INSIGHTS=true`):

| Variable | Description |
|---|---|
| `ENABLE_LLM_INSIGHTS` | Set to `true` to enable Gemini summaries |
| `GOOGLE_API_KEY` | Google Generative AI API key |
| `GENAI_MODEL` | Model name (default: `gemini-1.5-flash`) |
